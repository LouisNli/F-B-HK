# 客戶需求文檔 - 餐飲集團網站

## 專案概述

**客戶背景：** 香港一間小型餐飲公司，旗下擁有三個餐飲品牌

**專案目標：** 開發一個展示型企業網站，介紹集團旗下的三個餐飲品牌

---

## 功能需求

### 1. 網站架構

```
首頁 (Homepage)
├── 品牌一分頁
├── 品牌二分頁
└── 品牌三分頁
```

### 2. 頁面需求

#### 2.1 首頁 (Homepage)
- **內容類型：** 資訊展示 + 圖片為主
- **主要元素：**
  - 集團簡介
  - 三個品牌的快速導覽入口
  - 精選圖片展示
  - 聯絡資訊

#### 2.2 品牌分頁 (共三個)
- **內容類型：** 品牌專屬頁面
- **主要元素：**
  - 品牌介紹
  - 餐廳圖片展示
  - 菜單/特色菜式展示
  - 餐廳地址及營業時間
  - 聯絡方式

---

## 設計要求

| 項目 | 要求 |
|------|------|
| 整體風格 | 簡潔大方，不需過於花巧 |
| 視覺重點 | 以高質量圖片為主 |
| 響應式設計 | 支援桌面及移動裝置 |
| 用戶體驗 | 簡單直觀的導航 |

---

## 技術架構

- **前端框架：** Vue 3 + TypeScript
- **構建工具：** Vite
- **狀態管理：** Pinia
- **路由：** Vue Router
- **多語言支援：** i18n (中文/英文/葡文)

---

## 頁面路由規劃

| 路由路徑 | 頁面名稱 | 描述 |
|----------|----------|------|
| `/` | 首頁 | 集團主頁，資訊及圖片展示 |
| `/brand-1` | 品牌一 | 第一個餐飲品牌專頁 |
| `/brand-2` | 品牌二 | 第二個餐飲品牌專頁 |
| `/brand-3` | 品牌三 | 第三個餐飲品牌專頁 |

---

- [ ] 各品牌的 Logo 及品牌色彩
- [ ] 高解析度餐廳及菜式圖片
- [ ] 各品牌詳細資料（地址、營業時間、聯絡方式）
